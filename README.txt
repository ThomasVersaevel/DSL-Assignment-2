1. The HCL ecore metamodel is located in
    \nl.tue.dsldesign.hcl.metamodel\model\

2. The dynamic instances we have created are located in
    \nl.tue.dsldesign.hcl.metamodel\model\testModels\

3. The QVTo is located in
    \nl.tue.dsldesign.hcl2graph\transforms\HclTransformation.qvto

4. The resulting transformations after running the models from (2.) with qvto (3.) is located in
    \nl.tue.dsldesign.hcl2graph\result
    Here you can find the XMI file as well as a visual respresentation in svg format
